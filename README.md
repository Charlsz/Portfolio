## Cómo ejecutar el proyecto

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/mi-website.git
   ```
2. Accede al directorio:
   ```bash
   cd mi-website
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el entorno de desarrollo:
   ```bash
   npm run dev
   ```
